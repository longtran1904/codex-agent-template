# codex-agent-template

Inspired by [Self-Improving Coding Agents](https://addyosmani.com/blog/self-improving-agents/).

Goals:
- Build agent workflow that is autonomous for implementation, autonomous for research and generate issue tickets. The workflow should allow the agent to run and detect production/pipeline errors, generate an implementation ticket in `prd.json` where the tickets will be picked up and run by codex agent.
